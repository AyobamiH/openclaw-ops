# TOOLS - Normalization Agent

## Setup
```bash
cd agents/normalization-agent
npm install
```

## Testing
```bash
npm run test:local          # Quick tests
npm run test:performance    # Benchmark on large datasets
npm run validate:schema      # Verify output against target schema
```

## Monitoring
```bash
npm run check:quality       # Data quality metrics
npm run report:errors       # Error rate and failure analysis
```

## Cost
Per 100K records: ~$0.05  
Monthly (10M records): ~$50
