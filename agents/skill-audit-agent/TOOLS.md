# TOOLS - Skill Audit & Verification Agent

## Setup
```bash
cd agents/skill-audit-agent
npm install
```

## Testing
```bash
npm run test:local              # Quick tests
npm run audit:all               # Audit all skills
npm run test:regressions        # Check for regressions
npm run benchmark               # Run performance tests
```

## Monitoring
```bash
npm run check:compliance        # Security and code quality
npm run report:status           # Overall skill health
```
