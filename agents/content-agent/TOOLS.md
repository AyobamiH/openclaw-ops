# TOOLS - Content Generation Agent

## Setup
```bash
cd agents/content-agent
npm install
```

## Testing
```bash
npm run test:local              # Quick tests
npm run test:readme             # Generate test README
npm run test:api-docs           # Generate API documentation
npm run validate:completeness   # Check all features documented
```

## Monitoring
```bash
npm run check:examples          # Verify code examples are valid
npm run audit:coverage          # Feature documentation coverage
```

## Cost
Per 1000 words: ~$0.15  
Monthly (10 docs/month, 5K words each): ~$75
