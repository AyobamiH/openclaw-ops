# HEARTBEAT - Content Generation Agent

## Health Checks (Every 5 minutes)

✓ **Skill access:** documentParser available  
✓ **Generation speed:** <3 sec per 1000 words  
✓ **Output quality:** Well-formed markdown/HTML  

## Escalation
- 1st failure: Log warning
- 2nd failure (30 min): Log error
- 3rd failure (1 hour): Alert orchestrator
